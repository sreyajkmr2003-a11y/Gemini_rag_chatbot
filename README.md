# Gemini RAG Chatbot

## Project Title and Description

Gemini RAG Chatbot is an AI-powered question-answering system built using Retrieval-Augmented Generation (RAG).

It allows users to ingest website URLs and PDF documents, process the content, and ask questions based on the extracted information.

The system combines semantic search, vector similarity search, and large language model capabilities to provide accurate, context-aware responses using Groq Llama models.

---

# Features

- Website URL ingestion
- PDF ingestion support
- Multi-source RAG pipeline
- Semantic search using FAISS
- SentenceTransformer embeddings
- Groq Llama integration
- FastAPI backend
- React frontend
- Modern responsive chatbot UI
